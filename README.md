# 🎬 AI Hybrid Movie Recommendation System

## 🔥 Features
- Content-based movie recommendation
- Emotion-based recommendations
- Filters (duration, language, rating)
- Random movie suggestion
- Hybrid intelligent system

## 🧠 Tech Stack
- Python
- Pandas, Scikit-learn
- Streamlit

## 🚀 How it Works
1. Finds similar movies using cosine similarity
2. Matches user emotion with genres
3. Applies filters
4. Combines all results intelligently

streamlit run app/app.py
